## Area of a circle!

在编译环境中新建一个文件，输入以下代码并保存：

```python
import math

r = float(input("请输入圆的半径："))
area = math.pi * r ** 2
print("圆的面积为：", area)
```
运行结果
```bash
请输入圆的半径：114514
圆的面积为： 41197137648.525154
```
